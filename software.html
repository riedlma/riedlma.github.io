CoocViewer tool to visualize co-occurrences and positional co-occurrences
DRUID unsupervised method to rank n-grams regarding their multiwordness
JoBimText framework for distributional semantics, that incorporates methods for computing distributional thesauri, sense clusterings, IS-A patterns, etc.
JoBimViz visualization for the components of JoBimText. This visualization has an API that can be used for prototyping
SECOS unsupervised method for decompounding words. Models are provided for many languages.
TopicTiling unsupervised method for segmenting text into topical coherent units
